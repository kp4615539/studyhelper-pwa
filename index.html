Settings

Model must already be pulled via ollama pull gemma4:e2b. If this app is served from a different origin than Ollama, set OLLAMA_ORIGINS=* before starting the Ollama server.

Local encryption

Encrypts documents, notes, chat messages, flashcards and quizzes at rest inside this browser's storage (AES-256-GCM). Nothing is ever sent anywhere — this only protects data sitting on this device. If you forget your passphrase, encrypted data cannot be recovered.
